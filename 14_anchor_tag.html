앵커 태그

앵커 태그는 웹 페이지에서 다른 페이지로 연결하는 링크를 만든다.
앵커라는 단어는 배를 정박할 때 닻을 내려 고정하는 것과 유사하게 생각할 수 있다.
HTML에서 링크는 매우 중요한 요소이며, 링크를 설정할 때는 절대 경로와 상대 경로를 이해해야 한다.

절대 경로

절대 경로는 항상 동일한 경로를 가리키며, 아래의 네이버와 다음 링크가 그 예시이다.
앵커 태그는 <a> 태그로 시작하며, href 속성은 'hypertext reference'를 의미하고 링크할 URL을 지정한다.
링크를 클릭하면 해당 페이지로 이동한다. 'http' 방식은 'http://'로 시작하고, 보안 서버가 있는 경우 'https://'로 시작한다.
이러한 주소는 모두 절대 경로로 간주된다.

링크를 건 후 테이블의 사용법이라고 작성하고 저장한 후, 브라우저에서 테이블 링크를 확인하면 이전에 작성했던 테이블에 대한 내용이 나오는 것을 볼 수 있다.

상대경로 방식

링크 클릭 시 주의사항

링크를 작성할 때 주의해야 할 부분이 있다.
예를 들어, 사용자가 네이버 링크를 클릭하여 네이버 사이트를 보게 되면, 우리 페이지로 다시 돌아오는 방법이 사라질 수 있다.
따라서 외부 페이지의 경우에는 우리의 페이지에서 직접 열리는 것이 아닌 새 탭 또는 새 창으로 열려야 한다.
새 탭 또는 새 창으로 열리게 하려면, target 속성을 추가하고 값으로 _blank를 넣어야 한다.

target 속성 값

target에 넣을 수 있는 값은 다음과 같다.

  • _self: 기본값으로 현재 창에서 열린다.
  • _blank: 새 창이나 새 탭에서 열린다.
  • _parent: 부모 프레임에서 열린다.
  • _top: 전체 창에서 열린다.

기본적으로 기본 값은 _self이며, _blank는 새 창이나 새 탭으로 열리는 것이다.
새 탭으로 열리는 _blank만 기억하면 된다.
